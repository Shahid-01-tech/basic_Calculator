🧮 Calculator Web App

A simple and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations and is designed with a clean user interface.

🚀 Features

* Perform basic operations: addition, subtraction, multiplication, and division
* Responsive layout for desktop and mobile
* Interactive button inputs
* Clear and delete functionality
* Real-time display updates

🛠️ Technologies Used

* **HTML** – Structure of the calculator
* **CSS** – Styling and layout
* **JavaScript** – Functionality and logic

📂 Project Structure

```
calculator/
│── index.html
│── style.css
│── script.js
```

▶️ How to Run

1. Download or clone the repository
2. Open the `index.html` file in your browser
3. Start using the calculator

💡 How It Works

* User clicks buttons to input numbers and operations
* JavaScript captures input and updates the display
* When `=` is pressed, the expression is evaluated and the result is shown
